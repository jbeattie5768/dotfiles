Notepad++ notes
---------------

Default config file located at %appdata%\Notepad++, which resolves to C:\Users\<username>\AppData\Roaming\Notepad++

You can change to another location in Settings > Preferences > Cloud & Link > Cloud Config directory to some other location, e.g. P:\Temp.
If you want it to use AppData, remove the Cloud Config setting.
